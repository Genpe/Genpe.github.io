5、实现一个自定义网关过滤器：

  • 继承zuulFilter类

  • 重写shouldFilter，返回Boolean，设置是否执行过滤，true为执行，flase为不执行

    • 重写run，实现过滤器具体的业务逻辑

  • 重写filterType，返回过滤器类型，设置当前过滤器的类型（pre，route，post，error）

  • 重写order，返回int，数字越小越先执行

  • 查看路由端点

  • 过滤器端点
    访问：http://localhost:port/actuator/filters 可以查看默认的和自己定义的过滤器
